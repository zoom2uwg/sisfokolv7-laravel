## 1. TINJAUAN DOKUMEN 040: EPIC 9 (DEV REPORT KURIKULUM PLUGIN)

Laporan `DEV_DOCS-040` mencatat penyelesaian dan perbaikan bug untuk Plugin Kurikulum dengan test suite **3/3 PASS** (`KurikulumPluginTest`).

### 1.1 Hasil Analisis Bug & Solusi
* **Namespace Resolution**: FQN `\App\Support\PluginRegistry::class` digunakan untuk mengatasi kegagalan resolve class `PluginRegistry` relatif terhadap namespace testing.
* **Post-Boot Event Registration**: Mengatasi isu kegagalan subscriber event dengan memanggil `$events->subscribe()` secara eksplisit pada test suite. Ini dikarenakan event provider default Laravel tidak di-replay untuk class provider yang di-register secara dinamis saat runtime test.
* **Subscriber Handler Format**: Memperbaiki pemanggilan callback event subscriber dari `'methodName'` menjadi `'ClassName@methodName'`.

### 1.2 Kualitas UI/UX & Views
* views diletakkan secara modular di `app/Plugins/Kurikulum/Resources/views/` menggunakan namespace `kurikulum::`.
* Mengadopsi visual bertema premium dark-mode dengan Tailwind CSS dan Alpine.js yang konsisten dengan desain sistem utama.

## 2. TINJAUAN DOKUMEN 041: KESIAPAN ARSITEKTUR API-DRIVEN

Dokumen 041 meninjau realitas status arsitektur API pada SISFOKOL v7.

### 2.1 Temuan & Hambatan Utama
* **Aplikasi Saat Ini:** 99% masih bertumpu pada **Blade-SSR (Server-Side Rendering)**. Hanya ada 4 rute API di `routes/api.php` yang terintegrasi (Auth & Schedule).
* **Dipolaritas Database ("Parallel Universes")**: Terjadi divergensi skema antara tabel bahasa Inggris (`students`, `classrooms`, `subjects`) dan tabel bahasa Indonesia (`siswa`, `kelas`, `mapel`). Ini menyebabkan inkonsistensi data antar modul (misal: siswa baru di modul Akademik tidak muncul di penilaian).
* **Missing Infrastructure**: Folder `app/Http/Resources` belum dibuat di codebase nyata, hanya tertulis di rencana arsitektur. Konfigurasi `SANCTUM_STATEFUL_DOMAINS` di `.env.example` juga belum lengkap.
* **Ketergantungan State Session**: Sistem impersonation (Login-As) mengandalkan session cookies, yang tidak cocok untuk arsitektur API stateless.

| Area | Rekomendasi Tindakan | Target Fase |
